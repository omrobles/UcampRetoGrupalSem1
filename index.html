Etiqueta HTML Descripción
div Crear divisiones que nos permitan controlar mejor los elementos de la página.
h1 Resalta el texto con atributos específicos, mayormente usado en títulos.
p Inserter grandes cantidades de texto en el documento.
strong Resaltar parte un texto en negritas. También puede utilizarse la etiqueta "b".
ul Crear una lista de elementos no ordenada.
li Insertar elementos en la lista creada.
table Crear tablas en documentos HTML. Necesita de elementos th, td, tr, para construir la tabla.
